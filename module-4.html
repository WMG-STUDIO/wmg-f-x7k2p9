Module 4 · 20 minutes · le module le plus court

Héberger tes images, gratuitement

Petite étape indispensable : donner une adresse internet publique à tes images. Bonne nouvelle : ton compte GitHub du Module 1 fait ça gratuitement, en 3 clics.

Expliqué comme à un bébé
Quand ton facteur dit à Instagram « publie cette image », il ne peut pas lui tendre l'image : Instagram exige d'aller la chercher lui-même à une adresse, comme un livreur qui veut un point de retrait, pas un colis dans la rue. On va donc créer ton point de retrait : une étagère publique sur GitHub où l'usine déposera les images le temps qu'Instagram vienne les chercher.

4.1Pourquoi « public » et est-ce grave ?

L'étagère doit être publique (sinon Instagram ne peut pas venir chercher). Est-ce un problème ? Non : on n'y dépose que les images en cours de publication — des images qui, par définition, seront publiques sur ton profil quelques secondes plus tard. Aucun secret n'y passe jamais (jamais de token, jamais de mot de passe).

4.2Créer ton étagère (un « dépôt » public)

1
Sur github.com (connecté), clique le + en haut à droite → « New repository ».
menu + → New repository, fléché
2
Nom : mes-images · Visibilité : Public · coche « Add a README file » (ça évite une étagère 100 % vide) → Create repository.
formulaire de création : nom, Public coché, README coché
« Repository » ? « Dépôt » ?
C'est juste un dossier hébergé chez GitHub. Comme un dossier sur ton Bureau, mais accessible par internet. Rien de plus.

4.3Comprendre l'adresse publique d'une image

Testons à la main une fois, pour comprendre (ensuite l'usine le fera toute seule) :

1
Dans ton dépôt mes-images → « Add file » → « Upload files » → glisse ta slide du Module 2 (rendu/slide-modele.png) → bouton vert « Commit changes ».
glisser-déposer de l'image + bouton Commit changes
2
Clique sur le fichier dans la liste → clique le bouton « Raw ». Ton navigateur affiche l'image seule, avec une adresse qui ressemble à :
https://raw.githubusercontent.com/TON-COMPTE/mes-images/main/slide-modele.png
3
C'est ça, une adresse publique d'image. Copie-la, ouvre un onglet en navigation privée, colle-la : l'image s'affiche sans connexion → Instagram pourra venir la chercher exactement pareil.
l'image ouverte en Raw avec l'adresse surlignée dans la barre
Point de contrôle final du Module 4
  1. Mon dépôt public mes-images existe ✔
  2. Ma slide y est déposée, et son adresse « Raw » s'ouvre en navigation privée ✔
  3. Je peux expliquer pourquoi l'étagère est publique sans danger ✔
3/3 ? Ton point de retrait est ouvert. Au Module 5, on fait ta toute première publication automatique. 🚀

Glossaire du module — expliqué comme à un bébé

Dépôt (repository) 📁 — un dossier hébergé sur GitHub, accessible par internet.

Public / privé 👁️ — public : tout le monde peut regarder (pas modifier !). Privé : seulement toi. Nos secrets iront toujours dans du privé ; nos images en cours de publication, dans du public.

URL / adresse 🏷️ — l'adresse postale d'un fichier sur internet. Celle qui commence par https://…

Commit 📮 — le mot de GitHub pour « enregistrer ce dépôt de fichiers ». Chaque commit garde une trace datée (pratique : rien ne se perd jamais).

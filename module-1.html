Module 1 · 45 minutes · prérequis : Module 0 terminé

Créer tes comptes, à partir de rien

À la fin de ce module tu auras tes 4 comptes : l'email du projet, Instagram en version professionnelle, Facebook, et GitHub — plus l'application GitHub installée sur ton téléphone. On crée tout, dans l'ordre, depuis la page d'inscription.

Expliqué comme à un bébé
Aujourd'hui on fabrique tes clés. Une maison a besoin d'une porte d'entrée (ton email), d'une vitrine (Instagram), d'un badge à la mairie (Facebook/Meta) et d'un local pour tes employés invisibles (GitHub). Sans les clés, impossible de construire quoi que ce soit — alors on les fait toutes, proprement, maintenant.
Règle d'or du module
Note chaque mot de passe au fur et à mesure dans un carnet papier ou un gestionnaire de mots de passe. La moitié des abandons de débutants vient d'un mot de passe perdu 3 jours plus tard. Choisis des mots de passe différents pour chaque compte.

1.1L'adresse email dédiée au projet

Tout le projet vivra sur une seule adresse email. Si tu en as déjà une que tu consultes tous les jours, tu peux l'utiliser et sauter à l'étape 1.2. Sinon, on crée une adresse Gmail neuve :

1
Ouvre ton navigateur et va sur gmail.com → clique « Créer un compte » → « Pour le travail ou mon entreprise ».
page d'accueil Gmail avec le bouton « Créer un compte » fléché
2
Renseigne prénom, nom, puis choisis une adresse professionnelle : idéalement contact.tonentreprise@gmail.com ou tonentreprise.pro@gmail.com.
3
Google demande un numéro de téléphone : donne le tien, tape le code reçu par SMS. C'est aussi ta bouée de secours si tu perds le mot de passe.
écran de vérification par SMS
Point de contrôle 1.1
Ferme tout, rouvre gmail.com, connecte-toi avec ta nouvelle adresse. Tu vois ta boîte de réception vide ? Parfait. (Si tu n'y arrives pas, refais la connexion MAINTENANT — pas au Module 3.)

1.2Le compte Instagram

1
Sur ton téléphone, installe l'application Instagram (App Store sur iPhone / Play Store sur Android).
2
Ouvre-la → « Créer un compte » → inscris-toi avec l'email du projet (celui de l'étape 1.1 — pas ton email perso !).
écran d'inscription Instagram avec le champ email fléché
3
Choisis ton nom d'utilisateur : court, sans caractères bizarres, lisible à voix haute. Si ton premier choix est pris, ajoute un mot de ton métier plutôt que des chiffres (marie.fleuriste plutôt que marie93470).
4
Vérifie ton compte avec le code reçu par SMS ou email.

1.2 bis — Rendre le profil crédible (5 minutes qui changent tout)

Un profil vide fait fuir. Avant toute automatisation :

1
Photo de profil : ton logo si tu en as un, sinon une photo de toi propre et souriante. Va dans Profil → Modifier le profil → Modifier la photo.
2
Bio (3 lignes max) : qui tu aides + ce que tu publies + un lien. Exemple : « J'explique la compta aux artisans 🧮 · 1 conseil simple tous les 3 jours · ⤵ mon site ».
profil complété : photo, bio, lien — avant/après

1.2 ter — Passer en compte professionnel (obligatoire pour l'usine)

1
Profil → menu ☰ (en haut à droite) → Paramètres et confidentialité.
2
Descends jusqu'à « Type de compte et outils » → « Passer à un compte professionnel ».
chemin Paramètres → Type de compte → Passer à un compte professionnel (3 écrans)
3
Choisis « Entreprise » (pas « Créateur » — les deux marchent pour l'API, mais « Entreprise » simplifie la suite), puis une catégorie qui te ressemble.
4
Instagram te propose de connecter Facebook : « Ignorer » pour l'instant — on le fera proprement au Module 3, au bon moment.
Point de contrôle 1.2
Sur ton profil, tu vois maintenant ta catégorie affichée sous ton nom (ex. « Entreprise locale ») et un bouton « Tableau de bord professionnel ». Si oui → compte professionnel actif. ✔

1.3Le compte Facebook (ton badge d'accès à la mairie Meta)

Rappel du Module 0 : ce compte ne « postera » jamais rien. Il sert uniquement à obtenir les autorisations professionnelles de Meta. On le crée en 5 minutes :

1
Sur ton ordinateur, va sur facebook.com → « Créer un compte ».
2
Utilise ton vrai nom (Meta vérifie parfois l'identité — un faux nom peut bloquer ton usine plus tard) et l'email du projet.
formulaire d'inscription Facebook rempli (données d'exemple)
3
Confirme avec le code reçu par email. Tu peux ignorer toutes les suggestions d'amis — ce compte restera vide, c'est normal.
Si Facebook demande une photo ou une pièce d'identité
Ça arrive sur les comptes tout neufs (protection anti-robots — ironique, n'est-ce pas 😄). Fais la vérification demandée honnêtement : elle ne prend que quelques minutes et évite un blocage au pire moment.
Point de contrôle 1.3
Déconnecte-toi, reconnecte-toi sur facebook.com avec l'email du projet. Tu arrives sur ton fil (vide) sans question ? ✔

1.4Le compte GitHub (le local de tes employés invisibles)

1
Va sur github.com → « Sign up » (en haut à droite).
page d'accueil GitHub, bouton Sign up fléché
2
Email : celui du projet. Mot de passe : nouveau et différent. Nom d'utilisateur : simple, il n'apparaîtra nulle part publiquement (ex. marie-fleuriste).
3
GitHub envoie un code à 8 chiffres par email (« launch code ») : va le chercher dans Gmail, colle-le.
l'email GitHub avec le code + le champ où le coller
4
Aux questions « combien d'élèves dans ton équipe ? etc. », choisis « Just me » et le plan Free (gratuit). On n'aura jamais besoin de payer.
Point de contrôle 1.4
En haut à droite de github.com tu vois un petit rond (ton avatar). Clique dessus : ton nom d'utilisateur s'affiche. ✔

1.5L'application GitHub sur ton téléphone (ta télécommande)

C'est avec elle que tu diras « /publier » depuis ton canapé.

1
Installe « GitHub » (l'application officielle, logo chat noir) depuis l'App Store / Play Store.
2
Connecte-toi avec ton compte de l'étape 1.4.
3
Quand l'app demande à envoyer des notifications : ACCEPTE — c'est le messager de ton usine ! Sans ça, tu ne sauras pas qu'un carrousel attend ta validation.
app GitHub installée + popup de notifications avec « Autoriser » fléché
Point de contrôle final du Module 1
Fais l'inventaire de tes clés — tu dois pouvoir cocher les 5 lignes :
  1. Je me connecte à l'email du projet ✔
  2. Mon Instagram est en compte professionnel, avec photo et bio ✔
  3. Je me connecte à Facebook avec l'email du projet ✔
  4. Je me connecte à github.com sur l'ordinateur ✔
  5. L'app GitHub est sur mon téléphone, notifications autorisées ✔
5/5 ? Tes clés sont prêtes. Au Module 2, on crée ton identité visuelle.

Glossaire du module — expliqué comme à un bébé

Compte professionnel Instagram 🪪 — le même Instagram, mais avec la casquette « entreprise » : statistiques, bouton contact, et surtout le droit d'utiliser l'automatisation officielle.

Notification push 🔔 — le petit message qui s'affiche tout seul sur ton téléphone. C'est la sonnette de ton usine.

Gestionnaire de mots de passe 🔐 — une application coffre-fort qui retient tes mots de passe à ta place (ex. celui intégré à ton iPhone/Android). Mieux qu'un post-it.
